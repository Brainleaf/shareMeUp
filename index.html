Custom Button Style

You can customize all buttons simply adding a class called "custom" to the element you want customize. With "custom" class image will not charged and no style will be added to element. Using this options allows you to place your own button content inside element.

<span class="sharemeup custom" data-social="facebook">Custom Content</span>

Data Arrtibutes API

With shareMeUp you can specify some share parameters as: title, link, image and more...

Allowed data attributes

  • data-social="" Supported social networks' name
  • data-link="" Your post custom link
  • data-title="" Your post custom title Not work on Facebook and Google+ (use og tags)
  • data-media="" Custom image Only for Pinterest
  • data-domain="" Custom domain name / source Only for LinkedIn
  • data-custom="" If setted on "true" enables custom button feature
  • data-group="" If setted on "true" enables button groups feature

OG: Tags

ShareMeUp is compatible with "OpenGraphProtocol" tags.
To learn more about this protocol, please read the documentation at http://ogp.me/
